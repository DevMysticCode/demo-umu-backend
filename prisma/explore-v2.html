v2.3 · what's improved

What changed.

Personalised greeting + clearer page intent
Added a personalised welcome (avatar + "Good Afternoon, Alex") and kept the page's name as Explore — with a one-liner subtitle below it that explains what the page actually does. The label stays where users expect it; the subtitle does the work the bare "Explore" was missing.
Search splits into bar + tools
v1 stuffed input + distance + button into one tight pill. v2 gives the input room to breathe (rounded rectangle, larger tap target) with the distance and filter pills on a dedicated row underneath. A subtle live-counter on the right ("147 searched today") earns the page some warmth without taking action space.
Feature cards · in-between weight
Mid-point between the heavy block colour and the very light version. Each card carries: a 2px brand-coloured border, a soft tinted gradient background (teal-paler for HealthScore, navy-tinted for Passport), a filled brand-coloured icon tile next to the eyebrow, a filled brand-coloured pill on the right, an inset white "preview" panel for the gauge/book, and a filled brand-coloured CTA button. The page reads as branded and confident without the heaviness of full block-colour slabs.
"Watching" replaces "Pick up where you left off"
Returning users don't just want recency — they want signal. v2 shows HealthScore deltas (+3 pts, –2) and a teal "Updated" badge on properties with new data. The first card subtly tints teal when there's an update. The page becomes a useful dashboard, not just a history list.
Property cards · favouriting + ring
Added a heart button (one shown active, like a saved property), a sky gradient on each scene, and a tiny passport completeness ring next to "Passport · 100%" instead of just text. Photos now have a soft bottom gradient overlay so the address would still read if a real photo was used.
Tinted proof, tinted property hovers
The proof strip picks up a soft teal gradient and a 2px teal-pale border. Property cards now hover into a 2px teal border (instead of pale), and the "Passport · 100%" pill is filled teal with a soft glow. More colour shows up everywhere — but always as accent, never as full fill.
Bell with a quiet teal dot
Notification icon now carries a small teal dot indicator — nudges returning users towards the alert system the saved-search prompt is selling. Small detail, real lift.
Try it
Toggle New / Returning. Tap the heart on a property card. Drag the property and watching rows sideways. The "Updated" card on Returning shows what change-detection feels like as an everyday signal.
